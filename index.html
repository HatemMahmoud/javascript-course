History

1995

Mocha was created by Brendan Eich at Netscape, renamed LiveScript, and finally JavaScript.

1995

  • Client-side scripting with Netscape Navigator 2.0
  • Server-side scripting with Netscape Enterprise Server

1996

  • Client-side scripting with Internet Explorer 3.0
  • Server-side scripting with Internet Information Server
  • Microsoft's implementation was later renamed JScript

1997

First edition of the ECMAScript Language Specification (Standard ECMA-262)

2011

  • ECMAScript Edition 5.1
  • JavaScript version 1.8.5
  • JScript version 9.0

Today

  • All modern browsers fully support ECMAScript 5.1
  • JavaScript is officially managed by Mozilla Foundation

Object-Oriented
JavaScript

JavaScript Types

The simple types of JavaScript are:
numbers, strings, booleans, null, and undefined.

JavaScript Types

All other values are objects:
arrays, functions, regular expressions, and, of course, objects.

JavaScript Types

Numbers, strings, and booleans are object-like,

An object is a container of properties.

Working with Cookies

Cookies

A way to store and retrieve information on user’s computer to maintain state across requests.

Limitations

  • Strings only
  • 4KB per domain
  • Has an expiry date
  • Can be cleared by the user
  • Can be disabled by the user

Writing Cookies


document.cookie = "language=ar"; // expires after current session

Ajax

Asynchronous JavaScript + XML

A group of technologies used on the client-side to create asynchronous web applications.

Asynchronous JavaScript + XML

  • Presentation using HTML and CSS
  • Dynamic display and interaction using the DOM
  • Data interchange using XML
  • Asynchronous data retrieval using XMLHttpRequest
  • JavaScript binding everything together

XMLHttpRequest Object

Despite the name, the use of XML is not required (JSON is often used instead), and the requests do not need to be asynchronous.

XMLHttpRequest Methods


open(mode, url, boolean);   // creates a connection
// mode: GET or POST
// url: URL of the document
// boolean: true for asynchronous

send("string");             // sends a request to the server

XMLHttpRequest Properties


xhr.readyState          // 0: not initialized, 4: finished
xhr.status              // 200: OK, 404: not found, etc.
xhr.responseXml         // holds loaded data as XML
xhr.responseText        // holds loaded data as text
xhr.onreadystatechange  // takes a function

1. Create an instance


var xhr; 
try {
  xhr = new ActiveXObject('Msxml2.XMLHTTP');
}
catch (e) {
  try {
    xhr = new ActiveXObject('Microsoft.XMLHTTP');
  }
  catch (e2) {
    try {
      xhr = new XMLHttpRequest();     
    }
    catch (e3) {  
      xhr = false;   
    }
  }
}

2. Get ready for the response


xhr.onreadystatechange  = function() { 
  if(xhr.readyState == 4) {
    if(xhr.status == 200) 
      document.getElementById("response").innerHTML = 
      "Received: " + xhr.responseText; 
    else
      document.getElementById("response").innerHTML = 
      "Error: " + xhr.status;
  }
};
