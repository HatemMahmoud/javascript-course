History

1995

Mocha was created by Brendan Eich at Netscape, renamed LiveScript, and finally JavaScript.

1995

  • Client-side scripting with Netscape Navigator 2.0
  • Server-side scripting with Netscape Enterprise Server

1996

  • Client-side scripting with Internet Explorer 3.0
  • Server-side scripting with Internet Information Server
  • Microsoft's implementation was later renamed JScript

1997

First edition of the ECMAScript Language Specification (Standard ECMA-262)

Today

JavaScript has become one of the most popular programming languages on the web.
